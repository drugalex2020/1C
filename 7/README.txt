ОПИСАНИЕ
--------

Файл обработчика синхронизации номенклатуры для 1С:Предприятие 7.7 c PHPShop, поддерживается:

1. Комплексная конфигурация, редакция 4.5
2. Торговля+Склад, редакция 9.2


УСТАНОВКА
----------

Для запуска синхронизации 1С для 7,7 нужно выбрать  в меню программы 
"Файл" > "Открыть" > "Loader_Exchange.ert" из этого каталога.

При запуске "PHPShop_Exchange_77.ert" напрямую будет выводиться ошибка!


ПОЯСНЕНИЕ
----------

Файлы обработчиков для конфигураций "Бухгалтерия 7.7", "Упрощенная система налогообложения 7.7" 
находятся в подпапках "BUH" и "USN" соответствено.

Локализации для Украины и Казахстана находятся в подпапках "UKR" и "KZ" соответствено.

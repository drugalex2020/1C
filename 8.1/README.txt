ОПИСАНИЕ
--------

Файл обработчика синхронизации номенклатуры для 1С 8.1 c PHPShop, поддерживается:

1. Управление торговлей, редакция 10.3
2. Управление производственным предприятием, редакция 1.2
3. Управление торговлей для Украины, редакция 2.3
4. Управление торговым предприятием для Украины, редакция 1.0. 
5. Управление производственным предприятием для Украины, редакция 1.2


УСТАНОВКА
---------

Для запуска синхронизации 1С для 8,1 нужно выбрать  в меню программы 
"Файл" > "Открыть" > "PHPShop_Exchange_81.epf" из этого каталога.


ДОПОЛНИТЕЛЬНЫЕ УТИЛИТЫ
----------------------


1. PHPShop_Edit_81.epf - управление сайтом из 1С (новинка)
2. PHPShop_ImageLoad_81.epf - загрузчик изображений в базу 1с

Для запуска дополнительных утилит нужно выбрать  в меню программы 1С 
"Файл" > "Открыть" > "файл утилиты" из этого каталога.


ПОЯСНЕНИЕ
----------

Файл обработчика для конфигурации "Бухгалтерия 8.1" находятся в подпапке "BUH".

ОПИСАНИЕ
--------

Файл обработчика синхронизации номенклатуры для 1С 7.7 c PHPShop, поддерживается:

1. Комплексная конфигурация, редакция 4.5
2. Торговля+Склад, редакция 9.2


УСТАНОВКА
---------

Для запуска синхронизации 1С для 8,0 нужно выбрать  в меню программы 
"Файл" > "Открыть" > "PHPShop_Exchange_80.epf" из этого каталога.


ДОПОЛНИТЕЛЬНЫЕ УТИЛИТЫ
----------------------


1. PHPShop_Edit_80.epf - визуальный редактор описания для товаров
2. PHPShop_ImageLoad_80.epf - загрузчик изображений в базу 1с

Для запуска дополнительных утилит нужно выбрать  в меню программы 1С 
"Файл" > "Открыть" > "файл утилиты" из этого каталога.

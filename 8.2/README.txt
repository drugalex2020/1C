ОПИСАНИЕ
--------

Файл обработчика синхронизации номенклатуры для 1С 8.2 c PHPShop, поддерживается:

1. Управление торговлей, редакция 10.3
2. Комплексная автоматизация, ред. 1.0
3. Управление производственным предприятием, ред. 1.2, 1.3
4. Управление торговлей, редакция 11 
5. Розница, ред. 2.0
6. Управление небольшой фирмой, ред 1.3 
7. Управление торговлей для Беларуси редакция 3.0
8. Ювелирный Торговый Дом, редакция 10.4 


УСТАНОВКА
----------

Для запуска синхронизации 1С для 8.2 нужно выбрать  в меню программы 
"Файл" > "Открыть" > "Loader_Exchange.epf" из этого каталога и следовать инструкциям мастера запуска обработки обмена.

ОПИСАНИЕ
--------

Файл обработчика синхронизации номенклатуры для 1С 8.2 c PHPShop, поддерживается:

1. Управление торговлей, редакция 10.3
2. Комплексная автоматизация, ред. 1.0
4. Управление производственным предприятием, ред. 1.2, 1.3


УСТАНОВКА
---------

Для запуска синхронизации 1С для 8,2 нужно выбрать  в меню программы 
"Файл" > "Открыть" > "PHPShop_Exchange_82.epf" из этого каталога.


ДОПОЛНИТЕЛЬНЫЕ УТИЛИТЫ
----------------------


1. PHPShop_Edit_82.epf - управление сайтом из 1С
2. PHPShop_ImageLoad_82.epf - загрузчик изображений в базу 1с
3. PHPShop_ImportPriceSupplier.epf - загрузчик Excel прайсов поставщиков в базу 1С

Для запуска дополнительных утилит нужно выбрать  в меню программы 1С 
"Файл" > "Открыть" > "файл утилиты" из этого каталога.





ПОЯСНЕНИЕ
----------

Локализация для Украины находится в подпапке "UA", для Белоруси в подпапке "BY", для Казахстана в подпапке "KZ"
Конфигурация Ювелирный Торговый Дом, ред. 10.4 в подпапке "UVELIRSOFT".
Конфигурация Бухгалтерия, ред. 2.0 в подпапке "BUH".

Файлы обработчика для режима управляемого приложения находятся в подпапке "WEB".

1. Управление торговлей, редакция 11 (режим управляемого приложения до 11.0.6)  "WEB\TORG"
2. Управление торговлей, редакция 11 (режим управляемого приложения 11.0.7.13 и выше)  "WEB\TORG\11"
3. Розница, ред. 2.0 (режим управляемого приложения) "WEB\ROZNICA"
4. Управление небольшой фирмой, ред 1.3 (режим управляемого приложения) "WEB\FIRM"
5. PHPShop_Edit_82.epf - управление сайтом из 1С (режим управляемого приложения)
5. PHPShop_Monitor_82.epf - рабочее место оператора магазина 1С (режим управляемого приложения)

ОПИСАНИЕ
--------

Файл обработчика синхронизации номенклатуры для 1С 8.2 c PHPShop, поддерживается:

1. Ювелирный Торговый Дом, ред. 10.4 в подпапке "TD"
2. Ювелирный Салон, редакция 1.1 в подпапке "SALON"




УСТАНОВКА
---------

Для запуска синхронизации 1С для 8.2 нужно выбрать в меню программы 
"Файл" > "Открыть" > "PHPShop_Exchange_82.epf" из этих каталогов каталога.

ОПИСАНИЕ
--------

Файл обработчика синхронизации номенклатуры для 1С 8.2(режим управляемого приложения) c PHPShop, поддерживается:

1. Управление торговлей, редакция 11 (до 11.0.6 включительно)



УСТАНОВКА
---------

Для запуска синхронизации 1С для 8.2 нужно выбрать в меню программы 
"Файл" > "Открыть" > "PHPShop_Exchange_82.epf" из этого каталога.


ПОЯСНЕНИЕ
----------

Обработчик для конфигурации "Управление торговлей, редакция 11" (11.0.7.13) и выше находится в папке 11.
